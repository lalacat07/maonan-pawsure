# 毛安 PawSure · 微信小程序（按症状选粮 · 多平台比价）

宠物口粮**测评导购小程序**：按症状筛粮 → 看成分透明表与第三方检测 → 三平台自动比价 → 跳转/复制口令去对应平台购买。本站不经手交易。

## 如何运行
1. 下载并安装「微信开发者工具」。
2. 打开工具 → 导入项目 → 选择本文件夹 `maoan-miniprogram`。
3. AppID 可先选「测试号」体验；正式上线需注册小程序拿到自己的 AppID，填到 `project.config.json`。

## 目录
- `app.json` 全局页面/导航/底部 tab 配置
- `utils/data.js` 所有商品/症状/测评数据（**改这里就能改内容**）
- `pages/index` 首页（症状入口 + 热门）
- `pages/shop` 按症状筛选
- `pages/detail` 成分透明表 + 多平台比价 + 跳转
- `pages/reviews` 测评内容 + 买家反馈

## 重要：微信内的跳转限制（真实平台规则）
微信与阿里、拼多多是竞品，小程序**不能直接打开淘宝/拼多多**。本项目已按现实方案处理：
- **京东**：有官方小程序，`detail.js` 用 `wx.navigateToMiniProgram` 直跳。需在小程序后台「设置 → 第三方设置 → 跳转的小程序」配置京东 AppID，并填入 `detail.js` 的 `appId`。
- **淘宝/天猫、拼多多**：复制链接/口令到剪贴板 + 弹窗引导用户打开对应 App 粘贴购买（行业通用做法）。

## 上线前要替换的内容
- `utils/data.js` 里每个商品 `platforms[].target` 换成真实链接/淘口令/京东商品 path。
- 商品图标目前用 emoji，可换成真实图片（放 `images/`，WXML 用 `<image>`）。
- 注册小程序、完成主体认证；导购类目通常需相应资质，提交审核前确认类目要求。
